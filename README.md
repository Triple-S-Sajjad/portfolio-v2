## Local Development

```bash
git clone https://github.com/Triple-S-Sajjad/portfolio-v2.git
cd portfolio-v2
npm install
npm run dev
```

Open `http://localhost:3000`

---

## Deployment

Deployed on Netlify via GitHub integration.
Every push to `main` triggers an auto-deploy.

Build config in `netlify.toml`:
- Build command: `npm run build`
- Plugin: `@netlify/plugin-nextjs`
